# Vigeos Agent

Add-on Home Assistant de la box **Vigeos** : configuration automatique du kit
(activation par QR code depuis l'application), détection de chute exécutée sur
la box, envoi des alertes à la famille et chiffrement de bout en bout des
vidéos.

- Architectures : `aarch64` (Raspberry Pi 5, le kit Vigeos) et `amd64`.
- Distribué sous forme d'image déjà construite : `ghcr.io/lixisr/vigeos-agent-{arch}`.
